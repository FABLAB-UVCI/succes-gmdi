# Guide Utilisateur - Impression des Extraits d'Actes

## 🖨️ Comment imprimer un extrait d'acte?

### Pour les Naissances, Décès, Mariages et Certificats

#### Étape 1: Accéder au registre
1. Ouvrez le module correspondant (Naissances, Décès, Mariages ou Certificats)
2. Cliquez sur l'onglet **"Registre"** (ou le tab approprié)
3. Localisez l'acte que vous souhaitez imprimer

#### Étape 2: Cliquer sur le bouton d'impression
- Cliquez sur le bouton **<i class="ti ti-printer"></i> Imprimer**
- Une nouvelle fenêtre s'ouvrira automatiquement avec l'acte

#### Étape 3: Imprimer le document
**Option A - Impression directe:**
- La boîte de dialogue d'impression s'ouvre **automatiquement**
- Sélectionnez votre imprimante
- Cliquez sur **"Imprimer"**

**Option B - Enregistrer en PDF:**
- Dans la boîte de dialogue d'impression
- Sous "Imprimante", sélectionnez **"Enregistrer en tant que PDF"** (ou "Print to PDF")
- Cliquez sur **"Enregistrer"**
- Choisissez l'emplacement et confirmez

**Option C - Aperçu avant impression:**
- Vous pouvez visualiser l'acte avant d'imprimer
- Vérifiez les informations
- Ajustez les paramètres d'impression si nécessaire

## 📋 Contenu de l'extrait imprimé

Chaque extrait d'acte contient:
- ✅ En-tête officiel avec armoiries
- ✅ Logo du ministère de la Justice
- ✅ Numéro d'acte unique
- ✅ Tous les détails de l'acte
- ✅ QR code de vérification
- ✅ Signature numérique
- ✅ Mentions légales officielles
- ✅ Sceau et signature de l'officier

## 🎨 Paramètres d'impression recommandés

Pour une meilleure qualité d'impression:

| Paramètre | Recommandation |
|-----------|----------------|
| **Format de papier** | A4 |
| **Orientation** | Portrait |
| **Marges** | Par défaut (2.54 cm) |
| **Couleur** | Oui (pour le logo et le QR code) |
| **Qualité** | Haute |
| **Échelle** | 100% (ou "Adapter à la page") |

## 📱 Impression sur mobile

Sur un téléphone ou tablette:
1. Cliquez sur le bouton **Imprimer**
2. Attendez que l'acte s'affiche
3. Utilisez le menu du navigateur (≡)
4. Cherchez **"Imprimer"** ou **"Partager"**
5. Vous pouvez aussi:
   - Enregistrer en PDF
   - Envoyer par email
   - Imprimer via AirPrint (iPhone) ou Google Cloud Print (Android)

## 💾 Sauvegarder le document

### Option 1: Enregistrer en tant que PDF
1. Suivez les étapes d'impression
2. À la place d'imprimer, choisissez **"Enregistrer en PDF"**
3. Le fichier sera téléchargé

### Option 2: Capture d'écran
1. Ouvrez l'aperçu
2. Utilisez la capture d'écran de votre navigateur (F12 → Console)
3. Ou utilisez l'outil de capture du système

### Option 3: Exporter en JSON
Pour certains registres, un bouton **"Exporter Registre JSON"** peut être disponible pour télécharger toutes les données

## ⚙️ Dépannage

### Le document ne s'imprime pas

**Problem:** La boîte de dialogue n'apparaît pas
- ✅ Vérifiez que les pop-ups ne sont pas bloqués
- ✅ Autorisez les pop-ups pour ce site
- ✅ Essayez une autre navigateur
- ✅ Désactivez temporairement les extensions de navigateur

**Problem:** Le document s'affiche mal
- ✅ Vérifiez que JavaScript est activé
- ✅ Actualisez la page (F5)
- ✅ Essayez en mode incognito
- ✅ Videz le cache du navigateur

**Problem:** Les images (logo, QR code) ne s'affichent pas
- ✅ Vérifiez votre connexion Internet
- ✅ Attendez un peu (les images peuvent charger lentement)
- ✅ Contacter l'administrateur

### Problèmes de mise en page

**Si les marges sont incorrectes:**
1. Ouvrez la boîte de dialogue d'impression (Ctrl+P)
2. Cliquez sur **"Paramètres plus"** ou **"Paramètres supplémentaires"**
3. Ajustez les marges manuellement
4. Vérifiez que "Arrière-plan graphique" est **activé** (pour le logo et les bordures)

## 🔒 Sécurité et authenticité

Chaque extrait imprimé contient:
- 🔐 Un **code de vérification** unique (ex: 7A2B-C9D8-EF4G-H5J6)
- 📱 Un **QR code** scannabe pour vérification en ligne
- 🔒 Une **signature numérique** officielle

Pour vérifier l'authenticité:
1. Accédez à: **https://etatcivil.gouv.ci/verification**
2. Entrez le code de vérification
3. Ou scannez le QR code
